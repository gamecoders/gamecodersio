Video Games X-Plat con LibGDX

  • Sebastian Faltoni
  • Sun, Feb 22, 2015
  • libgdx
  • 0 Comment
Di recente sono venuto a conoscienza di LibGDX, una fantastica libreria per lo sviluppo di videogames cross platform in java. La buona notizia è che è completamente free per uso commerciale e non. Le piattaforme che supporta sono iOS, BlackBerry, Windows, Mac, Linux, Android e HTML5, tutte queste piattaforme sono raggiungibili utilizzando la solita code base, per quanto riguarda la parte HTML5, viene sfruttata la tecnologia GWT creata da google per trasformare il codice Java in Javascript.
